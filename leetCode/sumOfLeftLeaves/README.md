#Problem Description
Find the sum of all left leaves in a given binary tree.<br>
<br>
Example:<br>
<br>
    3<br>
   / \<br>
  9  20<br>
    /  \<br>
   15   7<br>

There are two left leaves in the binary tree, with values 9 and 15 respectively. Return 24.<br>
<br>
# Tips:
* Time complexity: O(n);
* Space complexity: O(1);
